Bu proje; kullanıcıların giyim, teknoloji, elektronik ve ev yaşam ürünlerini inceleyebileceği, Hepsiburada temalı modern bir E-Ticaret (Online Alışveriş) Platformunun Beta Sürümüdür. Sistem şu an prototip/geliştirme aşamasında olup, temel e-ticaret dinamiklerini ve yapay zeka entegrasyonunu test etmek amacıyla geliştirilmiştir.

Sistem, geleneksel listeleme sitelerinin ötesine geçerek kullanıcı deneyimini akıllı algoritmalarla simüle eder:

Gelişmiş Vitrin ve Ürün İnceleme (Beta): Kullanıcılar responsive (mobil uyumlu) bir arayüz üzerinde giyim ve teknoloji ürünlerini dinamik olarak filtreleyebilir, inceleyebilir ve sepetlerine ekleyebilir. (Not: Ödeme ve gerçek satın alım sistemleri henüz entegre edilmemiş olup, gelecek sürümlerde aktif edilecektir).

Yapay Zeka Destekli Akıllı Asistan: Sistemde yer alan gelişmiş yapay zeka/LLM (Anthropic Claude Sonnet 4.5) entegrasyonu sayesinde; demo kullanıcıların alışveriş eğilimleri ve teknoloji/giyim trendleri analiz edilerek kişiye özel akıllı ürün tavsiyeleri ve alışveriş simülasyon raporları sunulur.

Tam Yetkili Yönetim Paneli: Admin (Yönetici) kontrol paneli sayesinde; beta sürümünde yer alan tüm ürünlerin stok durumları, fiyatları, kategorileri (giyim, teknoloji vb.) dinamik olarak güncellenebilir ve sistem istatistikleri tam yetkiyle simüle edilebilir.
Sistem; dinamik sepet yönetimi, beğendiklerim (favori) listesi, admin paneli üzerinden tam yetkili kullanıcı/ürün yönetimi ve Anthropic Claude Sonnet 4.5 LLM entegrasyonu sayesinde araçların servis onarım geçmişlerini analiz ederek periyodik arıza raporları sunan yapay zeka destekli bir yönetim paneli barındırır.

🔒 v2.0 Güvenlik ve Mimari Yenilikleri:
Projenin backend ve veritabanı katmanlarında SQL Injection (SQLi) korumaları parametrize sorgularla tam olarak devreye alınmış, girdiler için sıkı bir input validation (veri doğrulama) mimarisi kurulmuştur. Modüller tamamen ayrıştırılarak test edilebilir, PEP 8 uyumlu ve kurumsal standartlarda bir yapıya kavuşturulmuştur.

🛠️ Kullanılan Teknolojiler ve Kütüphaneler
Backend Katmanı
FastAPI (Python): Hızlı, asenkron ve yüksek performanslı ASGI API mimarisi.

Uvicorn: Yıldırım hızında ASGI sunucu implementasyonu.

Motor & Pydantic: MongoDB için asenkron veri sürücüsü ve veri doğrulama (validation) kütüphanesi.

Keras & TensorFlow: Derin öğrenme tabanlı araç ve marka tanıma modeli (model_600_450_32_categorical.h5).

Tesseract OCR: Plaka karakterlerinin görsel üzerinden metne dönüştürülmesi.

Pytest & Bandit: %80+ test coverage (kod kapsama) oranı ve statik güvenlik taramaları (Security Scan) için.

Frontend Katmanı
React.js: Bileşen tabanlı, dinamik ve esnek kullanıcı arayüzü yönetimi.

Tailwind CSS & Lucide Icons: Modern, responsive (mobil uyumlu) Hepsiburada temalı görsel tasarım.

Sonner Toast: Kullanıcı aksiyonları için modern bildirim tetikleyicileri.

⚙️ Kurulum Adımları
1. Gereksinimler
Python 3.9+ veya üzeri kurulmuş olmalıdır.

Node.js (LTS sürümü) kurulmuş olmalıdır.

Tesseract OCR sistemi kurulmuş ve sistem PATH yoluna eklenmiş olmalıdır.

Yerel veya bulut tabanlı bir MongoDB sunucusu erişilebilir olmalıdır.

2. Bağımlılıkların Yüklenmesi
Terminal üzerinden projenin ana dizinlerine giderek kütüphaneleri yükleyin:

Bash
# Backend bağımlılıkları ve güvenlik kütüphaneleri için
cd backend
pip install -r requirements.txt

# Frontend bağımlılıkları için
cd ../frontend
npm install
📖 Çalıştırma ve Kullanım Talimatları
🟩 Backend Sunucusunun Başlatılması (Port: 8001)
FastAPI mimarisinin; veritabanı bağlantı adresleri, JWT gizli anahtarları ve Python bağımlılık yolları (PYTHONPATH) ile pürüzsüz çalışması için PowerShell terminaline aşağıdaki anahtar komut bloğunu yapıştırıp çalıştırın:

PowerShell
cd backend; $env:MONGO_URL="mongodb://localhost:27017"; $env:DB_NAME="aiservice"; $env:JWT_SECRET="test_gizli_anahtar_123"; $env:PYTHONPATH="$pwd\..;$pwd\..\.emergent"; python -m uvicorn server:app --port 8001
(Not: Eğer sisteminizde Docker yüklüyse, projeyi izole bir container mimarisinde tek tıkla kolay bir şekilde ayağa kaldırabilirsiniz.)

🟨 Veritabanı Verilerinin Yüklenmesi (Seed Data)
E-ticaret vitrininin ve demo araç yedek parça ürünlerinin MongoDB'ye otomatik yüklenmesi için backend klasörünün içinde şu scripti tetikleyin:

PowerShell
python veri_yukle.py
🟦 Frontend Arayüzünün Başlatılması (Port: 3001)
Ayrı bir terminal penceresinden Hepsiburada temalı e-ticaret arayüzünü canlı test modunda ayağa kaldırın:

Bash
cd frontend
npm start
🧪 Gelişmiş Test ve Güvenlik Taramalarının Çalıştırılması
Projenin kod kalitesini, SQL Injection korumalarını ve %80+ test kapsama oranını doğrulamak için backend dizininde şu komutları çalıştırabilirsiniz:

Bash
# Tüm birim ve entegrasyon testlerini koşturur
pytest

# Güvenlik açıklarını tarar (Bandit Security Scan)
bandit -r . -ll
